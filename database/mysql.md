
# Mysql


> MYSQL 数据库的优化方法

```
1. 选取最适用的字段属性,应该尽量把字段设置为 NOT NULL,在执行查询的时候不用比较 NULL 值。 
2. 使用连接(JOIN)来代替子查询(Sub-Queries) 
3 .使用联合(UNION)来代替手动创建的临时表
4 .尽量少使用 LIKE 关键字和通配符
5 .使用事务和外键

```
