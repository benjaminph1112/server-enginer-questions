
# 《服务端工程师题库》


## 缘起

笔试、面试是求职过程中非常重要的环节。很多同学能力不错，但却常常因为在笔试、面试时表现不好而与心仪的offer失之交臂；究其原因，不外乎是 基础早遗忘、考点没准备、原理说不清、细节犯迷糊。

本项目希望通过题库的形式，收集整理服务端工程师涉及的技能知识点，帮助求职者更顺利的找到好工作。
