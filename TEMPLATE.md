> ### 题目示例：请用5种以上的语言编写“hello world”程序

**标签：** `标签示例`,`标签2`

**解答：** 

*  php 

```php
<?php 
    echo 'hello world';

```

* javascript

```javascript
<script>
    alert("hello world")	
</script>

```

* golang

```go
package main 

import "fmt"

func main() {  
	fmt.Printf("hello, world\n") 
}

```

* c

```c
#include <stdio.h>

void main(void)
{
	printf("\nhello world!");
}

```

* java

```java
public class HelloWorld{ 
	public static void main(String[] args) {  
		System.out.println("Hello World!"); 
  	}
}
```
